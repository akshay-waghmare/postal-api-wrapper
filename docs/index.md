## What is MailIt Postal Wrapper?

MailIt Postal Wrapper provides a unified shipment tracking API over multiple carriers, with **India Post** as a primary supported courier.

Instead of integrating directly with India Post or third-party providers, clients use a single, stable API contract with:

*   Batch tracking support
*   Normalized statuses
*   Secure API keys
*   Rate limiting & plans
*   Clean audit trail
