# Normalized Statuses

| Status | Meaning (India Post) |
| :--- | :--- |
| `PENDING` | Tracking created |
| `INFO_RECEIVED` | Item booked |
| `IN_TRANSIT` | Dispatched / In transit |
| `OUT_FOR_DELIVERY` | Out for delivery |
| `DELIVERED` | Delivered |
| `EXCEPTION` | Delivery issue |
| `EXPIRED` | No updates |
| `NOT_FOUND` | Invalid tracking |
